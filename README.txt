# PowerShell Automation Portfolio – by Rajarshi Dwivedi

Welcome to my personal portfolio showcasing real-world automation utilities, scripts, and applications developed using **PowerShell**, **Python**, **Selenium**, and more.

These tools have been designed for professional use in domains like:
- Windows system automation
- API integrations
- Hotel tech operations
- LMS migrations
- Azure cloud service controls
- Script-based application backends

Each entry in the site includes:
- A script or utility name
- Its purpose and usage context
- Technologies used
- Key scripting concepts
- Download link or disabled placeholder for sensitive items
